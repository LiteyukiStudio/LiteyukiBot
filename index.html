    LiteyukiBot 6

    基于Nonebot2的OneBot标准聊天机器人，不仅仅局限于OneBot

    部署使用手册

    快速部署

    • 安装 Git 和 Python3.10+

    • 使用 git clone https://github.com/snowykami/LiteyukiBot 以克隆项目至本地。

      如果无法连接到GitHub，可以使用 git clone https://gitee.com/snowykami/LiteyukiBot。

    • 使用 cd LiteyukiBot 切换到项目目录。

    • 使用 pip install -r requirements.txt 安装项目依赖。

      如果你有多个 Python 环境，请使用 pythonx -m pip install -r requirements.txt。

    • 使用 python main.py 启动项目。
