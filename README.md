## 简介 Introduce

- 国际化，支持多语言，自定义语言

- 集成了一些美观且实用的功能，如插件管理，群聊管理，天气等

- 绝大多数配置操作可以通过QQ交互，避免对配置文件的手动修改

- 安装插件仅需通过QQ命令交互

- 稳定便捷的自动更新

- 轻量，少依赖，高稳定性

- 使用poetry构建虚拟环境，便于管理
