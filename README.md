## 关于

访问[轻雪6.0](https://bot.liteyuki.icu)主页获取更多信息

## 特点及优势

- 原生支持与任意`Python`Bot框架互联，有良好的生态支持
- 开箱即用，无需复杂配置
- 集成包管理器，支持一键安装插件
- 自定义主题支持，满足审美需求
- 国际化支持，支持多种语言
- 高性能，500插件2s内启动
