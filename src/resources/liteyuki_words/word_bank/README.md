# 这是什么 | What's this

一个特~~二刺螈~~（文爱）的适用于任何bot的词库<br>
好像有点涩？（不止一点

## 一些使用建议 | Using advices

- 词库返回中的“你”，可替换成目标昵称。同理，“我”也可替换成bot的昵称
- 如需使用分词，建议匹配无结果时将本词库key打乱，放回原句遍历判断
