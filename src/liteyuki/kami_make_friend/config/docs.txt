匿名交友
专治无对象，又想找对象的
命令：

- 寻找朋友
# 朋友为随机匹配，只能有一个

- 断绝朋友
# 绝交后可以交新朋友，但一个账号只有三次机会

- 连接朋友
# 连接后你发送的每一条消息都会转发给ta，ta的消息也会转发给你，但不能正常使用机器人，请注意不要触发违禁词

- 屏蔽朋友
# 突然心情不好，屏蔽ta，ta无法连接到你，但ta不知道
# 若想解除屏蔽，再发送一次即可

- 断开朋友
# 断开后机器人将进入正常模式

- 为朋友点歌 <歌名> (使用plat=xxx参数可以指定平台，详细请使用"help 音乐"查看)
# 给朋友点歌，并附上想对ta说的话

# 聊天时均为匿名，双方可以透露身份
